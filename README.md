# tanh

Calculate the hyperbolic tangent of a number.

## Example

``` javascript
const tanh = require('tanh');

tanh(1);
// => 0.7615941559557649
```

## Installation

``` bash
$ npm install tanh
```

## API

``` javascript
const tanh = require('tanh');
```

### `tanh(x)`

Given a _Number_ `x`, returns the hyperbolic tangent of `x` as a _Number_.
